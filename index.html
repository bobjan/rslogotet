Welcome to project pages.

The project is a contract bridge scoring application. It is aimed for bridge clubs with low budget, which can not afford expensive devices for the purpose. Currently, the project is at very start phase. The code available at GitHub is manly for the server engine. Needs a lot of refactoring. Android client is at visual design phase now. Hope to publich it very soon.

Special thanks to http://www.structure101.com/ whose product is of a very great help.
